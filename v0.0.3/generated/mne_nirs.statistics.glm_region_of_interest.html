mne_nirs.statistics.glm_region_of_interest¶

mne_nirs.statistics.glm_region_of_interest(stats, group_by, cond_idx, cond_name, weighted=True)[source]¶

Calculate statistics for region of interest.

Parameters
statsdict

Need to write.

group_bydict

Specifies which channels are aggregated into a single ROI. The dict key will be used as the ROI label and the dict values must be lists of picks (either channel names or integer indices of epochs.ch_names). For example:

group_by=dict(Left_ROI=[1, 2, 3, 4], Right_ROI=[5, 6, 7, 8])

Note that within a dict entry all channels must have the same type.

cond_idxinteger

Index of condition of interest.

cond_nameString

Name to be used for condition.

weightedBool

Should channels be weighted by inverse of standard error (True).

Returns
statsDataFrame

Statistics for each ROI.
